# LojaFácil — Sistema de Gestão para Lojas de Roupa

O **LojaFácil** é uma aplicação web voltada para auxiliar **lojas de roupas de pequeno e médio porte** no gerenciamento de produtos, clientes e vendas.  
A proposta é oferecer uma ferramenta simples, intuitiva e eficiente para otimizar o **controle de estoque**, reduzir **erros operacionais** e **melhorar o atendimento ao cliente**.

## Sobre o Projeto

Muitos lojistas ainda realizam o controle de estoque e vendas manualmente, o que causa **desorganização, perdas de produtos** e **dificuldade para acompanhar o desempenho das vendas**.  
O **LojaFácil** surge como uma solução acessível que automatiza essas tarefas, permitindo ao comerciante focar no que realmente importa: **vender mais e melhor**.

## Funcionalidades

- **Produtos:** cadastro, edição, exclusão e listagem de produtos  
- **Clientes:** cadastro e gerenciamento de informações de clientes  
- **Vendas:** registro de vendas e atualização automática do estoque  
- **Relatórios simples:** visualização de produtos com estoque baixo e histórico de vendas  

## Tecnologias Utilizadas

- **Node.js** — Ambiente de execução  
- **Express.js** — Framework backend  
- **Express-Handlebars (HBS)** — Template engine para renderização das páginas  
- **HTML5, CSS3 e JavaScript** — Frontend e interatividade  
- **Simulação de dados** em arrays (sem banco de dados real)

## Estrutura dos CRUDs

O projeto inclui três CRUDs principais:

1. **Produtos** — criação, leitura, atualização e exclusão  
2. **Clientes** — criação, leitura, atualização e exclusão  
3. **Vendas** — criação, leitura, atualização e exclusão (com atualização automática do estoque)
